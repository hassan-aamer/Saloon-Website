# Scarlett - Saloon & Spa Website

A modern, elegant, and responsive website designed for a Saloon and Spa business. This project showcases various beauty services, packages, and client testimonials with a seamless user experience.

## 🚀 Features

- **Fully Responsive Design**: Optimized for viewing on desktops, tablets, and mobile devices.
- **Interactive Navigation**: Sticky header with smooth scrolling links and a mobile-friendly hamburger menu.
- **Dynamic Services Carousel**: Interactive slider for displaying services, powered by SwiperJS.
- **Modern UI Elements**: Clean typography, high-quality imagery, and smooth CSS hover effects.
- **Sections Included**:
  - **Home**: Striking hero section with a clear call-to-action to book appointments.
  - **About Us**: Information about the salon's priorities, highlighting key offerings like Skin Care and Massages.
  - **Services**: Detailed breakdown of services (e.g., Dapper Facial, Hair Styling, Manicure) with pricing and duration.
  - **Testimonials**: Beautifully styled customer reviews.
  - **Footer**: Organized links, legal policies, contact info, and social media icons.

## 🛠️ Technologies Used

- **HTML5**: For semantic page structure.
- **CSS3**: Custom styling, animations, and responsive layouts (`main.css` & `responsive.css`).
- **JavaScript (Vanilla)**: For UI interactions like the mobile menu.
- **[SwiperJS](https://swiperjs.com/)**: Used for the touch-enabled services slider.
- **[BoxIcons](https://boxicons.com/)**: For lightweight, scalable vector icons.

## 🏗️ Getting Started

It is extremely easy to get this project up and running locally. No build tools are required.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hassan-aamer/Saloon-Website.git
   ```

2. **Navigate to the directory:**
   ```bash
   cd Saloon-Website
   ```

3. **Run the project:**
   Simply double-click the `index.html` file to open it in your default web browser. For developers, opening the project folder in VS Code and starting a **Live Server** is recommended for hot-reloading.
